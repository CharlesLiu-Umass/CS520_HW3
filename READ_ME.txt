Coverage_BEFORE_AND_AFTER - holds the coverage reports from before implementing the spec-guided tests and after. 
The reports are stored as html files that were directly exported from pytest-cov. 

Script - holds the python scripts used to evaluate the experiments done in homework 3. 
	GEMINI_generated_code - holds the code generated from Gemini when prompted back in homework 1
	GPT_generated_code - holds the code generated by GPT-5 when prompted back in homework 1
	test_unit - holds the unit test for both experiments. 
		The first 5 unit tests for both experiments are spec-guided while the rest are the original unit test from homework 1.
	test_gen_code.py - is the main file that imports the relavent files from the Script folder and when pytest is run, will generate a report

HW3_Report.pdf - the write-up for this homework that was also uploaded to GradeScope
